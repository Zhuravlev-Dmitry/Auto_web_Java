Тест-кейс 4. Удаление товара из корзины. 
Условия:
1. Пользователь успешно авторизован в системе.

3. Куки от предыдущего сеанса авторизации не удалены.
№	Шаг	Ожидаемый результат

4	Нажать на кнопку «Закрыть»	Страница закрыта.
